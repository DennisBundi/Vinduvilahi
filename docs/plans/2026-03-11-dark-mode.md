# Dark Mode Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a light/dark mode toggle with localStorage persistence for all users (customers and admins), defaulting to light mode.

**Architecture:** Use `next-themes` to manage theme state and inject a blocking inline script before paint (zero flash). Tailwind `darkMode: 'class'` strategy — `next-themes` toggles a `dark` class on `<html>`. A shared `ThemeToggle` icon button (sun/moon SVG) is placed in both the customer Header and the admin AdminNav top bar.

**Tech Stack:** next-themes, Tailwind CSS (darkMode: 'class'), React `useTheme` hook, localStorage

```ts
          dark: "#DB2777",
          light: "#F472B6",
        },
      },
    },
  },
  plugins: [],

```

### Task 2: Create ThemeToggle component

**Files:**
- Create: `src/components/ui/ThemeToggle.tsx`

**Step 1: Create the file**

Create `src/components/ui/ThemeToggle.tsx` with this exact content:

```tsx
'use client';

import { useTheme } from 'next-themes';
import { useEffect, useState } from 'react';

export default function ThemeToggle() {
  const [mounted, setMounted] = useState(false);
  const { theme, setTheme } = useTheme();

  useEffect(() => {
    setMounted(true);
  }, []);

  // Render nothing until mounted to prevent hydration mismatch
  if (!mounted) return null;

  const isDark = theme === 'dark';

  return (
    <button
      onClick={() => setTheme(isDark ? 'light' : 'dark')}
      className="p-2 text-gray-700 dark:text-gray-300 hover:text-primary dark:hover:text-primary transition-colors"
      aria-label={isDark ? 'Switch to light mode' : 'Switch to dark mode'}
      type="button"
    >
      {isDark ? (
        // Sun icon — shown in dark mode, click to go light
        <svg className="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
          <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2}
            d="M12 3v1m0 16v1m9-9h-1M4 12H3m15.364-6.364l-.707.707M6.343 17.657l-.707.707M17.657 17.657l-.707-.707M6.343 6.343l-.707-.707M12 8a4 4 0 100 8 4 4 0 000-8z"
          />
        </svg>
      ) : (
        // Moon icon — shown in light mode, click to go dark
        <svg className="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
          <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2}
            d="M20.354 15.354A9 9 0 018.646 3.646 9.003 9.003 0 0012 21a9.003 9.003 0 008.354-5.646z"
          />
        </svg>
      )}
    </button>
  );
}
```
